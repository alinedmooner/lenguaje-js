# Historia de JavaScript

JavaScript fue creado en Mayo de 1995, por **Brendan Eich** cuándo trabajaba en **Netscape**, implementando JavaScript en **Netscape Navigator**.

Existen dos estándar para JavaScript:

- ECMA-262
- ISO/IEC 16262

JavaScript también es llamado **ECMAScript** especialmente a partir de 2015.

## Comité técnico ECMA 39

Es el comité encargado de el proceso de adición de nuevas características del lenguaje, este comité está respaldado por empresas gigantes como: Adobe, Apple, Facebook, Google, Microsoft, Mozilla, Opera y Twitter.

El proceso que efectual el TC39 es 
- **0 strawman** Es el concepto inicial.
- **1 proposal** El TC39 interviene en el proyecto.
- **2 draft** Se comprueban las condiciones para su implementación.
- **3 candidate** Comienza un proceso de adquirir opiniones para mejoras.
- **4 finished** Listo para añadirse al lenguaje.
