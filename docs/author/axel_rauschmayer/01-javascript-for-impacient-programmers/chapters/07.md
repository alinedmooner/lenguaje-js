## Comentarios

Los comentarios son indicaciones puestas intencionalmente para proveer información del código.

En JS se pueden usar de la siguiente manera:
```javascript
// Variable que guarda el nombre del empleado.
const EMPLOYEE_NAME = "Daniel Moreno";
```

Los comentarios tienen notaciones especiales y normas para hacerlos más entendibles, se pueden hacer de muchas maneras pero eso depende del código de la apĺicación, proyecto etc.
una de las más usadas en **JSDoc**.
```javascript
/*
@param {number} amount quantity to tax.
*/
function addTax(amount) {
  return amount + amount*0.089;
}
```
