JavaScript tiene dos formas de uso:

- El navegador
- Node.js

Ambos cuentan con características distintas pero usan el mismo núcleo de JS.

En el navegador JS es utilizado principalmente para generar acciones de interacción, mientras que con node permite el uso de archivos, scripts y descargas vía http.

El entorno de JS consta de cuatro partes:

- La librería estándar de JS.
- Plataforma API.
- Motor de JS.
- Plataforma Núcleo.

JS cuenta con **npm** que actualmente es un sistema dominante que sostiene casi todos sus paquetes y librerías.
